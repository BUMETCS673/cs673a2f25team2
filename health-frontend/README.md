# 健康管理系统 - 前端

## 技术栈

- Vue 3
- Element Plus
- Vue Router
- Axios
- Pinia

## 安装依赖

```bash
npm install
```

## 启动开发服务器

```bash
npm run dev
```

## 构建生产版本

```bash
npm run build
```

## 预览生产构建

```bash
npm run preview
```

## 后端接口地址

默认后端地址：http://localhost:8080

如需修改，请在 `vite.config.js` 中修改代理配置。
