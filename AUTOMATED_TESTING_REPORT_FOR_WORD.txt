FRAMEWORKS & TOOLS USED

Backend Testing Stack
‚Ä¢ JUnit 5: Modern testing framework for Java
‚Ä¢ Mockito: Mocking framework for unit tests
‚Ä¢ Spring Boot Test: Integration testing support
‚Ä¢ Maven Surefire Plugin: Test execution and reporting

Commands:
Run all tests:
mvn test

Run with coverage:
mvn test jacoco:report

Run specific test class:
mvn test -Dtest=UserServiceTest

Frontend Testing Stack
‚Ä¢ Vitest: Fast unit test framework (Vite-native)
‚Ä¢ @vue/test-utils: Official Vue testing utilities
‚Ä¢ jsdom: Browser environment simulation
‚Ä¢ @vitest/coverage-v8: Code coverage reporting

Commands:
Run all tests:
npm run test

Run with coverage:
npm run test:coverage

Run with UI:
npm run test:ui

Watch mode:
npm run test -- --watch

Integration Testing
‚Ä¢ Postman: API endpoint testing
‚Ä¢ Newman: CLI test runner for Postman collections

Commands:
Run Postman collection:
newman run postman-collections/User-API-Tests.json --environment env/local.json
